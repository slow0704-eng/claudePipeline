# 목 데이터 빠른 사용 가이드

## 🚀 빠른 시작

### 1단계: 목 데이터 삽입

프로젝트 루트에서 다음 명령 실행:

```batch
insert_mock_data.bat
```

### 2단계: 옵션 선택

실행하면 다음 화면이 표시됩니다:

```
==========================================
기존 데이터 삭제 여부 선택
==========================================

기존 데이터가 있으면 중복 오류가 발생할 수 있습니다.

1. 기존 데이터 삭제 후 삽입 (권장)
2. 기존 데이터 유지하고 삽입

선택하세요 (1 또는 2):
```

**권장**: `1` 입력 → Enter

### 3단계: 애플리케이션 실행

```batch
mvnw.cmd spring-boot:run
```

### 4단계: 테스트

브라우저에서 http://localhost:8080 접속

**로그인 정보**:
- 사용자명: `user001` (또는 user002~user050)
- 비밀번호: `1234`

## ⚠️ 문제 해결

### 중복 오류 발생 시

```
ERROR: duplicate key value violates unique constraint
```

**해결 방법**:

```batch
clear_mock_data.bat
```

실행 후 다시 `insert_mock_data.bat` 실행

### PostgreSQL 연결 오류

1. PostgreSQL 서비스가 실행 중인지 확인
2. `application.properties`의 DB 설정 확인:
   ```properties
   spring.datasource.url=jdbc:postgresql://localhost:5432/boarddb
   spring.datasource.username=postgres
   spring.datasource.password=1
   ```
